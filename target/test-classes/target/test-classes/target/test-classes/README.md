## fabric-sdk-java 实现案例

### 简单测试流程

#### 1、启动网络

使用如下命令clone[代码库](https://github.com/ldstyle8/fabCar.git):

```
$ https://github.com/ldstyle8/fabCar.git
```

使用如下命令启动fabric底层网络：

```
$ cd cd network
$ ./build.sh
```

后续完成测试后可以使用如下命令关闭或清理fabric网络环境：

```
$ ./stop.sh
```

或者

```
$ ./teardown.sh
```
